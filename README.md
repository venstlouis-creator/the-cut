# The Cut 🏆

Personal weight loss tracker — OMAD · 20K steps · Strength 4×/week

## Features
- Weight logging with history
- Progress toward June 7 goal
- Weekly checkpoint timeline
- Workout tracker (strength + light days)
- Meal logger with food chips
- Meal ideas based on your foods

## Run locally
```bash
npm install
npm start
```

## Deploy to GitHub Pages
```bash
npm run build
# Push the build/ folder or use gh-pages package
```

## Stack
- React 18
- Recharts
- localStorage (no backend needed)
